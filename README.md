# VivaTEQ Building Envelope Thermal Calculator — Public Trial Demo

This public repository contains the limited VivaTEQ trial demo for Singapore building-envelope thermal calculations.

## Trial scope

Available in the public trial:

- ETTV calculations across eight façade orientations
- RETV calculations and weighted building results
- validated BCA-linked standards manifest

Disabled in the public trial:

- RTTV
- U-Value tools
- drawing upload and file storage
- Excel, PDF, Word and print exports

The private full-product source is maintained separately under VivaTEQ.
